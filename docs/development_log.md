## 🗓️ Day 17：ストア公開に向けた最終調整
### 19. ストア掲載文のローカライズ思考
*   **課題**: どのように書けば「ただのTodoアプリ」と差別化できるか。
*   **解決**: 「常駐通知」と「プライバシー（サーバー送信なし）」を強調し、シンプルさを求めるユーザーに刺さる言葉を選定。

### 20. アプリアイコンの互換性
*   **課題**: 機種によってアイコンの形がバラバラになる問題。
*   **解決**: Android Studio の Image Asset ツールを使い、前景・背景の2層構造を持つ「Adaptive Icon」を作成。Resize機能を使って、どのマスク（丸・四角）でも見切れないように調整。
