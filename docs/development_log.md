## 🗓️ Day 18：Google Play Store 審査提出
### 21. 正式な Application ID への移行
*   **課題**: 練習用の `com.example` ではストア公開ができない、または推奨されない。
*   **解決**: `com.todaymemo.myapp` へ一括変更。単なる ID 変更にとどまらず、ソースコードのパッケージディレクトリ構造（Folder Structure）を物理的に再構築し、ビルドエラーをすべて解消した。
### 22. バージョンアップ・ポリシー
*   **課題**: ストアへのアップロード時に「以前と同じバージョン」では受け付けられない。
*   **解決**: `versionCode` を 2、`versionName` を 1.1 へ更新。リリースサイクルにおける管理方法を実践した。
